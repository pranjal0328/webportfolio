Homelessness Impact
Analysis in San Diego

As a Research Assistant at SDSU, I am collaborating on an NSF Funded project to develop a deep learning model that detects homelessness encampments using Google Street View images. We initially created a self-annotation tool for manually labeling encampments, which was then used to train the model for automated detection. Our goal is to develop an interactive map visualizing encampments across San Diego using recent GSV data, conduct migration analysis, and provide data-driven insights to inform and enhance urban policy recommendations.

Instagram Influencer Score Analysis Model

The Instagram Influencer Score Analysis Model predicts the effectiveness of Instagram influencers by analyzing their engagement metrics and performance. Using logistic regression and decision tree classification, the model processes and cleans influencer data, then assigns a score based on predictive insights. Car Sales Price Prediction Model

Car Sales Price Prediction Model is a machine learning project aimed at predicting the price of used Toyota Corolla cars based on various specifications such as age, mileage, fuel type, and features. The model uses a neural network with different hidden layer configurations, trained on a dataset of 1,436 records. Data preprocessing includes feature scaling using MinMaxScaler and encoding categorical variables with pandas. The model is evaluated using RMSE to compare training and validation performance across different architectures, helping to determine the optimal network configuration for accurate price prediction.
